# Vagrant PHP7 

A simple Vagrant LAMP setup running PHP7.

## What's inside?

- Official Ubuntu 16.04 LTS (Xenial Xerus)
- Nginx
- PHP7 with some extensions
- MariaDB
- Composer

## How to use

- Clone this repository into your project
- Run ``vagrant up``
- Navigate to ``http://192.168.33.100/``
